Welcome to the MVT Toolkits! This document will demonstrate how to use this tool to conduct MVT and how to read the result.

How to use this Tool:

	1). Open MVT_app.R in R and run the script. you should see a window popping up. 
	2). In the window, you are asked to upload the file and input the desired significance level. Please upload the file and input sigfinicant level as instructed. 

How to read the result:

	There are two tabs in the window. the first one reports on the P-value of ANOVA, the seconde reports on detailed variants pair of siginicant difference(if any).
